# [WIP] nyumbu-ci

Action based on VM

## Get started

1. Clone the repo
    - `git clone https://github.com/nyumbu/nyumbu-ci.git`

2. Start web
    - `cd nyumbu_web`
    - `bun install`
    - `bun run dev`

3. Start server
    - `cd nyumbu_server`
    - `pdm install`
    - `pdm run start`

4. Open http://localhost:5173/#/workflow/
